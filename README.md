# Gemini 斗地主 (Web 局域网联机版)

基于 React + PeerJS + TailwindCSS 开发的高颜值斗地主游戏，支持局域网无服务器联机。

## 🌟 核心功能

1.  **即点即玩**：无需下载，浏览器打开即可。利用 P2P 技术，只需输入房间号即可联机。
2.  **专用服务器 (观战) 模式**：
    *   非常适合**聚会**场景。
    *   电脑作为主机创建“专用房间”，投屏到电视上。
    *   3个朋友用手机扫描加入，手机作为手柄，电视直播战况（支持上帝视角显示所有手牌）。
3.  **癞子玩法**：可选择开启“天地癞子”玩法的简化版（随机一张癞子牌，可作为万能牌）。
4.  **智能辅助**：
    *   **滑动选牌**：手机端专属优化，像原生 App 一样流畅。
    *   **AI 参谋**：接入 Gemini API，分析牌局给你出牌建议。
    *   **机器人托管**：三缺一？一键添加机器人凑数。

## 📱 移动端优化

*   支持 **Touch 滑动选牌**：手指滑过即可选中/取消选中多张牌。
*   **布局自适应**：根据手牌数量动态调整重叠密度，防止溢出屏幕。
*   **防误触**：操作区域按钮增大，并在非操作时隐藏。

## 🛠️ 技术栈

*   **React 18**: UI 构建
*   **PeerJS**: WebRTC 封装，实现浏览器点对点通信
*   **TailwindCSS**: 现代化的原子类 CSS 框架
*   **Google Gemini API**: 提供 AI 游戏建议

## 🚀 快速开始

1.  打开网页。
2.  **房主**：点击“创建房间”。
    *   如果是想自己打，选“创建房间 (我来打牌)”。
    *   如果是投屏观战，选“创建专用房间”。
3.  **其他玩家**：输入房主的 ID（屏幕上会显示），点击“加入房间”。
4.  人齐后，房主点击“开始游戏”。

## 🎨 素材与音效

项目内置了纯 CSS 绘制的精美卡牌，支持替换为图片。音效需自行添加至 `public/assets/audio` 目录。
